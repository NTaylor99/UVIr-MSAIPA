The three files here are the data sets of irradiance values generated from the modified TUV Irradiance Model. Each file is the data set for one of the three locations used for the study: El Paso, TX (elp003.txt); Flagstaff, AZ (flg010.txt); Logan,UT (log004.txt).
The format of each file follows the standard default output table for an unmodified TUV source code with custom inputs, and the time intervals are set for 3 min increments with 80 total data points for each individual table. The output tables then repeat in the same format beginning at the time stamp immediately proceeding the end of the previous table. There are a total of 18 total output tables in each file, resulting in 1440 data points for each location.
